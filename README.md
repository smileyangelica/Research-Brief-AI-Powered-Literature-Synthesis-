# In-Space Manufacturing & Space Biotech Research Brief

## Overview
This repository contains a comprehensive research brief on in-space manufacturing with a focused section on space biotechnology. The brief synthesizes current scientific understanding of microgravity-based production systems and their commercial applications.

## Contents
- **index.html** - Main research brief webpage
- **_config.yml** - GitHub Pages configuration
- **.gitignore** - Git exclusion file

## Deployment on GitHub Pages

### Instructions
1. Create a new GitHub repository (or use an existing one)
2. Clone the repository locally
3. Copy these files to the repository root:
   - `index.html`
   - `_config.yml` (optional, for Jekyll configuration)
   - `.gitignore`

4. Push to GitHub:
   ```bash
   git add .
   git commit -m "Add space biotech research brief"
   git push origin main
   ```

5. Enable GitHub Pages in repository settings:
   - Go to Settings → Pages
   - Set source to "Deploy from a branch"
   - Select "main" branch and "/ (root)" folder
   - Save

6. Your site will be published at: `https://{username}.github.io/{repo-name}/`
